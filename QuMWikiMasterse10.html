Barplots and Histograms

The following parts of code deal with differences between barplots and histogram. Barplot can be used to compare numerical data divided into different groups. Histograms visualize the probability of a variable to lie in a certain interval. Having the number of breaks of an interval going to infinity gives a density. Keep in mind that the probability is the area of a certain bar and not only its height.

#' When to use which of the following plots
?hist
?barplot
#' Draw 5'000'000 sizes (heights of persons) out of a normal distribution,
#' with mean=175 and sd=7.5.
myMean <- 175
mySD <- 7.5
set.seed(2603)
heights <- rnorm(5000000, mean=myMean, sd=mySD)
hist(heights, breaks=30, col=rgb(0,0,1,1/4))
#' col ... color
#'   rgb ... values for red, blue, green and a transparancy factor alpha in [0,1]
set.seed(2603)
hist(
  rnorm(5000000, mean=myMean-5, sd=mySD+1),
  breaks=60, add=TRUE, col=rgb(1,0,0,1/4)
)
#' add ... if plot is added to the last plot (working for some plot functions)
myAlpha <- 0.02
(myLowerBound <- qnorm(myAlpha/2, mean=myMean, sd=mySD))
## [1] 157.5524

(myUpperBound <- qnorm(1-(myAlpha/2), mean=myMean, sd=mySD))
## [1] 192.4476

abline(v=myLowerBound, col="blue")
abline(v=myUpperBound, col="blue")

Usage of stringsAsFactors

Usually, it is more convenient to read data (/or to create a data frame) as character strings. This is done via stringsAsFactors=FALSE. After having manipulated the data for further use it is advisable to typecast the data to types as needed.

#' Use "stringsAsFactors" and then after processing the data,
#' type cast it as wanted.
names <- c("Hans", "Max", "Eva", "Adam", "Anton", "Barbara", "Carl", "Maximilian")
set.seed(2603)
size <- rnorm(8, mean=myMean, sd=mySD)
