3.3 Windows

3.3.1 “Pure” R (Windows)

First go to https://cran.r-project.org/, select “Download R for Windows”, click at “install R for the first time”, and choose “Download R x.x.x for Windows” [Note: Note: x.x.x stands for the current version (3.5.1 at the date of authoring).]   . Save the file and run it. Depending on your chosen language preferences there can be other expressions in the boxes.

You should to confirm the license agreement, then you have to choose your installation location.

Depending on your operating system and your hardware, in the next step
either the 32-bit files or the 64-bit files have to be chosen.


If you want to have a folder in your Windows Start Menu, it has to be named now.

In the next step additional parameters can be chosen. Presence of desktop icon, symbol in your task bar, saving the version ID in your registry and linking *.RData-files to R. It is recommended to choose at least the last two options.


Now you have completed the installation process of R. Open R and test simple calculations to notice whether it operates (correctly).


3.3.2 RStudio (Windows)

The next step to take is to download RStudio.

Go to https://www.rstudio.com/products/rstudio/download/\#download and choose “RStudio x.x.xxx - Windows Vista/7/8/10” [Note: Note: x.x.xxx stands for the current version (1.1.456 at the date of authoring).]   . Save the file and run it. Depending on your chosen language preferences there can be other expressions in the boxes.

As a next step the file destination has to be chosen.


If you want to have an entry in your Windows Start Menu, it has to be named now.

After the automatic installation you have reached the end of the RStudio installation process. Please open RStudio and check if it operates by doing simple calculations.


3.3.3 MiKTeX (Windows)

Go to https://miktex.org/download/, select the Windows-tab, click at Installer-tab, and choose “Download” [Note: Note: The file name contains the version ID (2.9.6850-x64 at the date of authoring).]   . Save the file and run it. Depending on your chosen language preferences there can be other expressions in the boxes.

In a first step it is recommended to accept the conditions of use. Secondly it can be chosen whether you want to install it only for yourself or for all users.


The path, where MiKTeX will be installed has to be determined. Two settings have to be set in a further step: The preferred paper size to A4 and “Install missing packages on-the-fly”. In order to have no problems in the first compilation of the first report, it is recommended to set this parameter to Yes. This can be changed later on.


The following window gives an overview of your settings. The installation process can take a while.


Now MiKTeX wants to check for updates, which is in general positive and should be done. If not, open the MiKTeX console and check for updates there (and install them). In the end the tick to learn more about donation can be removed, unless you want to donate.
