3.4 Apple

3.4.1 “Pure” R (Apple)

Mac user face a variety of different versions depending on their OS. You can see the latest releases and in the description the suitable OS is stated. After downloading the installer (*.pkg file), it should look similar to the picture below.


In the next two steps the “Read Me” and the license agreement are displayed.


We recommend to agree the terms of the software license in order to use it. The installation type can be chosen now.


After the successful installation process you are asked to delete the installation file.


3.4.2 RStudio (Apple)

The next step to take is to download RStudio.

Go to https://www.rstudio.com/products/rstudio/download/\#download and choose “RStudio x.x.xxx - Mac OS X 10.6+ (64-bit)” [Note: Note: x.x.xxx stands for the current version (1.1.456 at the date of authoring).]   , if your OS X is newer or equal Mac OS X 10.6. Save the file and run it. Depending on your chosen language preferences there can be other expressions in the boxes.

Drag the icon to your applications. It is recommended to agree to the licence agreement.

Then the installation process should run automatically.


Please open RStudio and check if it operates by doing simple calculations.

3.4.3 (MiKTeX/) MacTeX (Apple)

It is recommended to install MacTeX, if you are working with an Apple device, because Mac users may face problems using MiKTeX. This website [Note: https://www.reed.edu/data-at-reed/software/R/r_studio.html]   gives a step-by-step installation guide. The R-packages do not have to be installed by hand in modern versions, i e. section 8 in the online-guide can be skipped. RStudio will check if all necessary packages are installed and will ask if you want to have them installed automatically when creating your first report.
