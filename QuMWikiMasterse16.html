3.1 R – Installation guide

In the course Quantitative Methods 1 you will get familiar with the front end programming language R as well as you will learn generating reports containing R output using MiKTeX [Note: Note: This guide will be extended by MiKTeX as soon as possible.]   . Therefore the graphical environment RStudio is recommended. The following sections will guide you through the installation process. Please also search the internet for other installation guides because this one does not claim to be exhaustive. If you have doubts at any point whether you want to accept the license agreements, you have the possibility to work on the computers provided in the LC or TC. If your installation does not work, this is no problem, since computers with R and RStudio are available for students in the LC and TC.
