2.6 QuM2 – Session 6 (Hypothesis Testing)

Let the body heights of the people in the population be normal distributed with μ = 175 and σ = 7.5.

mu.population <- 175
sigma.population <- 7.5

In the following, hypothesis testing is presented.

Steps to conduct hypothesis testing

  1. Formulation of H0 and HA.
  2. Specifying the significance level α.
  3. Constructing the rejection region.
  4. Collecting the data.
  5. Determining the sample size n.
  6. Calculating the test statistics (and p-values).
  7. Decision (reject or do not reject H0).
  8. Interpretation of the result.

Formulation of H0 and HA

The formulation of the Null Hypothesis H0 can be done in many ways, the most popular of it is stating that H0: Θ = Θ0. The altenative hypothesis HA is often the logical negation of H0, i. e. HA: Θ≠Θ0.

Specifying the significance level α

Specifying the significance area implies specifying the rejection interval (and consequentely the accenptence [MAINTAIN] region).

Therefore, a short excourse to the two main types of errors is given:

There exist two types of errors, i. e. type I and type II errors, sometimes called α- and β-error according to their probabilities.

Table 2.2: Type I and type II errors
Reality
H0 HA
Decision Fail to reject H0 OK (True negative) Type II error with probability β
Be able to reject H0Type I error with probability α OK (True positive)

As a next step, for constructing the rejection region, the significance level α is specified. This implies specifying the type I error. Usually α ∈ (0, 0.1), e. g. α := 0.05.

alpha <- 0.05

According to α, the critical value(s) can be calculated.

Constructing the rejection region

This critical values lead to the rejection region. Calculating critical t-values is only possible after detrmining the sample size n, since it is used to calculate the degrees of freedom, which is a parameter of the student’s t-distribution.

Table 2.3: Calculating critical values depending on sample size and knowledge of sigma
sample size

large

small

σ isknown

take the known sigma and the quantile of the normal distribution

take the known sigma and the quantile of the normal distribution

unknown

take the sample standard deviation and the quantile of the normal distribution due to CLT

take the sample standard deviation and the quantile of the student’s t-distribution

The critical values are the quantiles of the corresponding distribution. Using the student’s t-distribution, the sample size decreased by one (for calculating σ), i. e. n − 1, is the number of degrees of freedom.

#' sigma is known OR the sample is large
(z.alpha <- qnorm(alpha) )
## [1] -1.644854

Collecting the data and determining the sample size n

For explaination purpose, instead of collectig data, two samples are simulated.

# set.seed(2305)
sample.n.large <- 1000
sample.n.small <- 20

sample.large <- rnorm(sample.n.large, mean=mu.population, sd=sigma.population)
sample.small <- rnorm(sample.n.small, mean=mu.population, sd=sigma.population)

Now it is possible to calculate the critical values for small samples, when the population mean is unknown because the degrees of freedom are known.

#' sigma is unknown, sample is small
(t.alpha <- qt(alpha, df= sample.n.small - 1) )
## [1] -1.729133

In this example critical values for lower one-sided (/tailed) tests have been calculated.

Examples for upper one-tailed tests and two-tailed tests are given:

#' sigma is unknown, sample is small
#' critical value for upper one-tailed test
(t.oneminusalpha <- qt(1 - alpha, df= sample.n.small - 1) )
## [1] 1.729133

#' critical values for two-tailed test
(t.alphahalf <- qt(.5 * alpha, df= sample.n.small - 1) )
## [1] -2.093024

(t.oneminusalphahalf <- qt(1 - (.5 * alpha), df= sample.n.small - 1) )
## [1] 2.093024

Obviously, due to the symmetry of the normal and the student’s t-distribution,

PX ≤−x = PX > x = 1 −PX ≤ x.

Having normally distributed data is essential for testing. The larger the data sets are, the “better” the tests gonna be.

Calculating the test statistics (and p-values)

For small and large normally distributed data sets with known variance σ2,
the test statistic is calculated by z = x̄−μ0 σ∕n ;
For … large normally distributed data sets with unknown variance σ2,
the test statistic is calculted by z ≈x̄−μ0 s∕n ;
For small … normally distributed data sets with unknown variance σ2,
the test statistic is calculted by t = x̄−μ0 s∕n ;
where x̄is the arithmetic mean of the observations,
μ0is the population mean,
σis the variance of the population,
sis the sample variance
and nis the number of observations.

According to the formulas and the different cases, the test statistics are calculated.

#' sigma is known(, sample is large)
(
  z.stat.large.known <-
    (mean(sample.large) - mu.population) *
    sigma.population^(-1) * sqrt(sample.n.large)
)
## [1] -1.337909

#' sigma is known(, sample is small)
(
  z.stat.small.known <-
    (mean(sample.small) - mu.population) *
    sigma.population^(-1) * sqrt(sample.n.small)
)
## [1] -0.2267274

#' sigma is unknown, sample is large
(
  z.stat.large.unknown <-
    (mean(sample.large) - mu.population) *
    sd(sample.large)^(-1) * sqrt(sample.n.large)
)
## [1] -1.32961

#' sigma is unknown, sample is small
(
  t.stat.small.unknown <-
    (mean(sample.small) - mu.population) *
    sd(sample.small)^(-1) * sqrt(sample.n.small)
)
## [1] -0.3293311

Decision (reject or do not reject H0)

Finally, the test statistic is compared to the critical t-value t0 and the p-value is compared to α.

The two most popular instruments for making a decision to reject or to not reject H0 are summarized in Table 2.4:

Table 2.4: Decision about H0 and HA
stay with H0reject H0
t-, z-statistic |tstat|≤|tα| |tstat| > |tα|
p-value pval ≥ α pval < α
(t.stat.MUplus4 < t.alphahalf) || (t.stat.MUplus4 > t.oneminusalphahalf)
## [1] TRUE

p.value.test1 < alpha
## [1] TRUE

Indeed, both conditions are true, so the null hypothesis H0 can be rejacted and the alternative hypothesis HA is taken to be true (until it is falsified).

Interpretation of the results

Consequentliy the difference in the means is significantly different from zero. Perhaps we have wrong assumptions about our population, e. g. μ≠175, or the sample is not drawn out of the original population with μ = 175 and σ = 7.5.

Various examples

Test of a Population Mean

Suppose we have a sample of size n = 40 of heights of people in a country. The sample mean is computed as x̄ = 164cm and it is known that the population standard deviation is σ = 20. A researcher believes that the average height of the population is less than 175cm. Test researchers’ claim at α = 0.05 significance level.

First the two hypotheses are formulated:

H0:μ = 175cm
HA:μ < 175cm

Consequentely we conduct a one-sided test.

n.sample <- 40
x.bar <- 164
sigma.population <- 20
mu.population <- 175
alpha <- 0.05
z.alpha <- qnorm(alpha)

The critical value for α = 0.05 is zα = −1.6448536, using $z_\alpha = \Sexpr{z.alpha}$.

The R code for the test statistic is given by the following chunk.

(
  z.statistic <-
    (x.bar - mu.population) * sigma.population^(-1) * sqrt(n.sample)
)
## [1] -3.478505

z.statistic < z.alpha
## [1] TRUE

pnorm(z.statistic) < alpha
## [1] TRUE

In the code chunk above, first

z = x̄ − μ σ∕n = 164 − 175 20∕40 = −3.4785054

and then z < zα. Additionally, the p-value is calculated and compared to α.

Indeed, a significantly big difference in body heights is observed, so the null hypothesis H0 is rejected and it is believed that the alternative hypothesis HA is true (until it is falsified). So that researchers’ claim at α = 0.05 significance level is supposed to be true.

Test of Hypothesis about a Population Proportion

ABC Company produces manufacturing items. Suppose 300 items are randomly selected from a very large shipment, each is tested and 10 defective items are found. Does this outcome provide sufficient evidence for the production company to conclude that the fraction of defective items in the entire shipment is less than 0.05? Take α = 0.01.

Like it was mentioned in the session on confidence intervals the random variable is Bernoulli distributed.

favourable.cases <- 10
possible.cases <- 300
p.population <- 0.05
alpha <- 0.01
z.alpha <- qnorm(alpha)
p.hat <- favourable.cases / possible.cases
sigma.hat <- sqrt(p.hat * (1 - p.hat))

Therefore,

p̂ = 10 300 = 0.0333333

is a suitable estimateor for p.

The variance is given by Vp = p(1 − p) and thus, σ = p(1 − p).

First the two hypotheses are formulated:

H0:p = 0.05
HA:p < 0.05

Consequentely we conduct a one-sided test.

The critical value for α = 0.01 is zα = −2.3263479.

The R code for the test statistic is given by the following chunk.

(
  z.statistic <-
    (p.hat - p.population) * sigma.hat^(-1) * sqrt(possible.cases)
)
## [1] -1.608169

z.statistic < z.alpha
## [1] FALSE

pnorm(z.statistic) < alpha
## [1] FALSE

In the code chunk above, first

z = p̂ − p σ∕n = 0.0333333 − 0.05 0.1795055∕300 = −1.6081688

and then z < zα. Additionally, the p-value is calculated and compared to α.

Accordingly to the sample, the evidence that p < 0.05 is not strong enough. So the null hypothesis H0 is kept.
