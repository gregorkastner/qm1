1.5 QM1 – Session 5 (Distributions)

Using sample to roll the dice, generating data

First, to have a certain starting point, a seed has to be set. From this point on, the results are reproduceable, but (pseudo) random. This means that before doing it once, the outcome is not predictible by most of the people.

set.seed(20190816)  # sets the seed to the number 20190816

The sample command draws ”size” many times out of the first input parameter, which is a vector

sample(1:6)  # draws one time out of the vector 1:6, which is equal to rolling a die
## [1] 4 6 3 2 1 5

sample(1:6, size=10, replace=TRUE)  # draws 10 times and returns it afterwards
##  [1] 6 1 1 2 6 5 5 2 3 2

sample(1:6, size=10, replace=TRUE, prob=c(1, 1, 2, 2, 3, 3))
##  [1] 6 4 5 2 2 6 6 5 6 6

#' probabilities can be added for every element of the first vector
table(sample(1:6, size=24, replace=TRUE))  # counts how often a number appeared
##  
## 1 2 3 4 5 6  
## 8 1 1 3 7 4

Using sample Distributions and the d/p/q/r-functions

With the help of R many distributions can be calculated. In particular the ones listed:

?Distributions

For most of the distributions the d/p/q/r-functions are available. In Table 1.2 an overview can be found.

Generating random normals

Another function can draw random numbers of the normal distribution.

amountRandomNormNr <- 1000  # assigns the number of draws to be done
myrandomnorm <- rnorm(amountRandomNormNr, mean = 1, sd = 2)  # draws 1000 numbers
random.norm <- myrandomnorm  # "random.norm" gets the numbers
#' In order to use "table", the numbers are grouped
           (random.norm < -5)[1:10]  # returns the first ten values of a TRUE/FALSE vector
##  [1] FALSE FALSE FALSE FALSE FALSE FALSE FALSE FALSE FALSE FALSE

random.norm[random.norm < -5] <- -5.5  # -5.5 gets assigned to all numbers < -5
random.norm[(random.norm >= -5) & random.norm < -4] <- -4.5
random.norm[(random.norm >= -4) & random.norm < -3] <- -3.5
random.norm[(random.norm >= -3) & random.norm < -2] <- -2.5
random.norm[(random.norm >= -2) & random.norm < -1] <- -1.5
random.norm[(random.norm >= -1) & random.norm <  0] <- -0.5
random.norm[(random.norm >=  0) & random.norm <  1] <-  0.5
random.norm[(random.norm >=  1) & random.norm <  2] <-  1.5
random.norm[(random.norm >=  2) & random.norm <  3] <-  2.5
random.norm[(random.norm >=  3) & random.norm <  4] <-  3.5
random.norm[(random.norm >=  4) & random.norm <  5] <-  4.5
random.norm[(random.norm >=  5) & random.norm <  6] <-  5.5
random.norm[random.norm >= 6] <- 6.5  # 6.5 gets assigned to all numbers >= 6
plot(table(random.norm)/amountRandomNormNr)
lines(xs, dnorm(xs, mean = 1, sd = 2), col = 2)
