Install packages e. g. lpSolve

Keep in mind to install and load R packages before using them.

Packages can be installed via install.packages(pkgs, ...). As it is stated in the documentation of the command [?install.packages], pkgs needs a “character vector of the names of packages whose current versions should be downloaded from the repositories.” Therefore, install.packages("lpSolve") can be used.

install.packages(lpSolve) will not work out of the box. However, after assigning lpSolve <- "lpSolve" it can be used. ;)

If any questions occur, do not hesitate to read the documentation.
