Glossary

arrows
The function arrows adds arrows to an existing plot. It has the following main input parameters:
positionname

description

example

1x0

(vector with) x-coordinates of starting points from which to draw the arrows

c(0,5)

2y0

(vector with) y-coordinates of starting points from which to draw the arrows

c(1,5)

3x1

(vector with) x-coordinates of end points to which to draw the arrows

c(2,1)

4y1

(vector with) y-coordinates of end points to which to draw the arrows

c(3,1)

barplot
The function barplot creates a barplot. It has the following main input parameters:
positionname

description

example

1x

numerical vector or matrix containing values for which a barplot should be plotted. Every column of a matrix is one unit of bars.

rbind(1:3, 4:6)

beside

logical value, if the columns of of the input matrix should be plotted beside each other or stacked

FALSE by default

horiz

logical value if the bars should be drawn horizontally or vertically

FALSE by default

constrOptim
The function constrOptim minimises a function subject to linear inequality constraints using an adaptive barrier algorithm.

The system of equations of constraints reads like this:

MYMATRIX

It has the following main input parameters:

positionname

description

example

1theta

numeric vector containing p initial values. Those must be in the feasible region.

c(6,6)

2f

function to minimise

profit

3grad

gradient of f

NULL

4ui

constraint matrix (k × p)

matrix(c(1, 0, 0, 1, -1, -1, 1, 1), 4, 2, byrow = TRUE)

5ci

constraint vector of length k

c(0, 0, -12-1e-5, 12-1e-5)

control

List of different control parameters, in particular:

list(fnscale = -1, reltol= 1e-8)

Should for an argument maximizing the function be searched?

fnscale = -1

Relative convergence tolerance. The algorithm stops if it is unable to reduce the value by a factor of reltol * (abs(val) + reltol) at a step. Defaults to sqrt(.Machine$double.eps), typically about 1e-8.

reltol=1e-8

contour
The function contour creates a countour plot or adds contour line to an existing plot. It has the following main input parameters:
positionname

description

example

1x

optional parameter that describes the x-location of the z coordinates. Usually it is provided as the rows of the grid.

c(1:100)

2y

optional parameter that describes the y-location of the z coordinates. Usually it is provided as the columns of the grid.

c(1,100)

3z

parameter that is the result of having the function evaluated at the points of the grid spanned by x and y

c(100,100)

4nlevels

optional parameter that states the number of contour lines

10

positionname

description

example

profits

2from

0

3to

end point of the range over which the function will be plotted

100

xname

The function curve3d is the analogue of curve, taking a function with two input parameters. Thus it creates a surface and plots it. Before using sys3d="persp", the rgl-package has to be installed via install.packages("rgl"). It has the following main input parameters:
positionname

description

example

1expr

function to be plotted (containing 2 input parameters)

profits

2from

starting point of the range over which the function will be plotted

c(0,0)

3to

starting point of the range over which the function will be plotted

c(100,100)

varnames

vector of character strings giving the names to be used for the independent axes

c("Q1", "Q2")

sys3d

3D plotting system to use

one of "persp", "wireframe", "rgl", "contour", "image", or "none"

lp
The function lp optimizes a linear function subject to linear inequality constraints. It has the following main input parameters:
positionname

description

example

1direction

Character string giving direction of optimization.

Either "min" by default or "max"

2objective.in

Numeric vector of coefficients of the linear objective function

c(50, 30)

3const.mat

constraint matrix (see constrOptim)

rbind(c(1, 0), c(0, 1), c(2, 1), c(5, 5), c(1, 3))

4const.dir

Vector of character strings giving the direction of the constraint

c(" >=", " >=", " <=", " <=", " <=")

5const.rhs

constraint vector on the rhs of the equation

c(0, 0, 14, 40, 18)
